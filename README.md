# 🎓 Coursework Copilot — Powered by ProjectMate

**Coursework Copilot** is an AI-powered web app built under the **ProjectMate** ecosystem, designed to help university students plan, track, and complete their coursework efficiently.  
It leverages intelligent document parsing, automated task planning, and progress tracking to ensure students stay organised and aligned with their project specifications.

---

## 🚀 What the Project Does

Coursework Copilot enables students to:
- 📄 **Upload coursework specifications** (PDF, DOCX, or plain text).
- 🤖 **Use AI (via ProjectMate)** to extract key details:
  - Deliverables (e.g. report, code, presentation)
  - Deadlines and milestones
  - Assessment criteria and weighting
  - Constraints or special requirements (e.g. “Use Python”, “Max 2000 words”)
- 🗂️ **Automatically generate structured plans**, with subtasks, estimated effort, and suggested schedules.
- 📊 **Track progress** using a dashboard that visualises task completion, deadlines, and overdue items.
- 🔔 **Receive smart insights and alerts**, such as:
  - “Your implementation section is incomplete.”
  - “Deadline in 3 days — you’re 20% behind schedule.”

---

## 💡 Why the Project Is Useful

Managing multiple coursework deadlines can quickly become overwhelming. Coursework Copilot simplifies this by:
- ✅ Turning ambiguous project specs into clear, actionable tasks.
- ⏱️ Helping students balance workloads across weeks or semesters.
- 📬 Providing proactive reminders before critical deadlines.
- 🧠 Highlighting missing deliverables or underdeveloped sections using LLM feedback.
- 🎯 Keeping users focused and confident that they’re meeting all project requirements.

Whether for dissertations, coding assignments, or group projects, Coursework Copilot helps students *work smarter, not harder.*
